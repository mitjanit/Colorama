# Colorama
<h2>Gestió de Color</h2>
<p>Conjunt de funcions per a la gestió de color.</p>
<ul>
	<li>Crea paleta de colors aleatòria basada en imatge.</li>
	<li>Crea paleta de colors espaiada basada en imatge.</li>
</ul>
